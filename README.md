# Task Manager API

## О проекте

Task Manager API — REST-приложение для управления задачами, разработанное на Java. Проект предоставляет API для создания, просмотра, обновления и удаления задач, а также демонстрирует применение современных технологий разработки серверных приложений.

## Основные возможности

- Создание задач
- Получение списка задач
- Просмотр информации о задаче
- Обновление существующих задач
- Удаление задач
- Валидация входных данных
- Работа с приоритетами и статусами задач

## Используемые технологии

- Java
- Spring Boot
- Spring Web
- Spring Validation

- Maven

## Запуск проекта

### Клонирование репозитория

```bash
git clone https://github.com/Goslinuly/java-final.git
```

### Переход в директорию проекта

```bash
cd java-final
```

### Сборка проекта

```bash
mvn clean install
```

### Запуск приложения

```bash
mvn spring-boot:run
```

После запуска приложение будет доступно по адресу:

```
http://localhost:8080
```

## API

Пример основных REST-запросов:

| Метод | Endpoint | Описание |
|-------|----------|----------|
| GET | `/tasks` | Получить список задач |
| GET | `/tasks/{id}` | Получить задачу по ID |
| POST | `/tasks` | Создать задачу |
| PUT | `/tasks/{id}` | Обновить задачу |
| DELETE | `/tasks/{id}` | Удалить задачу |

## Модель задачи

Каждая задача содержит:

- Название
- Описание
- Статус
- Приоритет
- Срок выполнения
